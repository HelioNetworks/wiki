# .NET

HelioHost offers the following .NET versions: 
* .NET 1.1
* .NET 2.0
* .NET 3.0
* .NET 3.5
* .NET 4.0
* .NET 4.5
* .NET 4.6
* .NET 4.7
* .NET 5.0
* .NET 6.0
* .NET 8.0
* .NET Core 3.1
* Classic ASP

To use these, you would need to request an account on [Lily](../servers/virtual/lily.md).
